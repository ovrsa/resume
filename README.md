### 1.4. 要約

4年間、セイアローズにて特注家具製作のディレクターとして活動。<br>
プロジェクトによってオーナーやデザイナーが求める要素は大きく異なるため、お客様が心地よく過ごせる場所をイメージし、国内外の工場のメリットを活かした家具の企画‧設計‧製造‧販売を中心に包括的に管理をしてきた。<br>
お客様が求めるスタイルやトレンドに合わせながら、常に高品質かつコストパフォーマンスに優れた製品を提供した。<br>
その他にも、メンバーの業務効率を上げるためにアプリを開発し、社内に導入なども行った。

レアジョブテクノロジーズでは、コーポレートエンジニアとして、レアジョブグループ全体の従業員の生産性向上を目指し、各部署のワークフローを最適化する企画から設計・開発・運用を経験。<br>
所属チームにツールを導入することで業務改革にも貢献し、ツールの導入、汎用処理のライブラリ開発、コードレビューのルール化など、開発環境の改革にも貢献した。
主に(JavaScript/Python)を用いた社内向けシステム開発における、要件定義からフロントエンド・バックエンド開発、DB設計など幅広く担当した。<br>

### 1.5.2. 2019/04 - 2023/08 : 株式会社セイアローズ
特注家具製作のディレクターとして、プロジェクトの進捗管理、予算管理、技術的検討、製造パートナー選定、仕様設計など、アイデアを具体的な形にするまでの包括的な管理を担当。

#### プロジェクト管理

- **進捗と予算管理**: プロジェクト全体の進捗と予算を管理し、プロジェクト目標の達成に向けて全体を調整。顧客のニーズを理解し、目的に応じた目標設定と課題の細分化を行うことで、プロジェクトの効率化と品質向上を実現。

- **要求への対応**: 要求に対しては常に全体像を意識し、計画段階から逆算して行動。クライアントの依頼内容を正確に把握し、目的達成のために注力。具体的には、クライアントのニーズと目標を理解し、使用環境を詳細にイメージしながら計画を策定。100%の要求達成を基本とし、さらにその上を目指すアクションを議論し、無駄な行動を排除して具体的なゴールを設定。

- **計画と分析**: 目標に向けた手順や計画を慎重に設計し、問題点や必要なアクションを詳細に分析。このアプローチにより、プロジェクト全体を俯瞰し、問題を早期に特定してプロジェクトのスムーズな進行を実現。

#### 技術的検討と製造パートナーの選定

- **技術検討**: 製品の仕様や設計の技術的側面を評価し、品質とコストのバランスを最適化。
- **製造パートナーの選定**: 主に中国企業から適切な製造パートナーを選定し、プロジェクトの成功に寄与。

#### 顧客要望の実現

- 顧客のニーズに基づき、アイデアを具体的な製品に変換し、実現可能な設計を提案。ヒアリングから設計、製造に至るまで、関連する全てのステークホルダーと連携し、スムーズなコミュニケーションを通じて問題解決を実施。

#### プロジェクト以外の実績

- **新プロダクトの開発**: OEMプロジェクトで新しいタイプのチェアを開発。
  - プロモーションから製造、販売まで一貫して担当し、製品を市場に投入。

- **業務用アプリ開発**: 社内全体の業務効率を向上させるための業務用アプリを開発。
  - Excel VBA、GAS、RPA（SikuliX）を活用し、ルーチン作業を自動化。生産性と効率性を向上。
  - **製作事例メール送信システム**: 手作業の工数を削減するため、製作事例メール送信システムを開発。
  - **データ管理アプリの開発**: FlaskとFirestoreを使用して社内用会計データ管理アプリを開発。手動で行っていたデータの集計・確認作業を自動化し、作業時間を短縮し、ミスの削減を達成。

- **Google Workspaceの導入**: 業務運用の効率化を目的にGoogle Workspaceを全社導入。業務のデジタル化とコミュニケーションの効率化を推進。
